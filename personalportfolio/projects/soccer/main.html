Soccer match data processing tool

URL: http://www.lategoals.com/

The first phase of this project is complete and has been handed over to the customer.

Note: Due to the sensitivity of the company's data, I have no credentials to provide for login.

Data Capture

  • Developed code to automatically capture and store data from a series of specified websites that provide soccer match data on a regular basis.
  • Processed the data previously captured using specially developed algorithms as outlined by the client.
  • Implemented the back end data capture side of the project in a modular format using Python. Used unit testing to ensure the integrity of the code.

Graphical Display of Data

  • Created a web based application to allow the client to view the processed data.
  • Developed a JSON structure to fetch back data from the server to the web application.
  • Data is fetched back every couple of seconds and at peak times, a high number of soccer matches can take place simultaneously creating large data packets.
  • Used Javascript over jQuery to make frequent DOM updates to the web application without any stuttering or flickering.
  • Added the ability for website users to create and log into their personal accounts to customize various aspects of the website and save their settings.
